# SETU.AI — Intelligent AI Tutoring Platform
A full-stack AI tutoring platform built with RAG (Retrieval-Augmented Generation), 
enabling contextual question answering from structured and unstructured knowledge sources.

## 🚀 Live Demo
🌐 [Try SETU.AI Live](https://setuai-five.vercel.app/login)

## 🛠️ Tech Stack
- **Backend:** Spring Boot, LangChain, OpenAI/Groq APIs, FAISS, RAG Pipeline
- **Frontend:** React.js, Tailwind CSS, Lip-sync Animation
- **Database:** MySQL, Vector Search (FAISS / Pinecone)

## 📁 Repositories
- 🔗 [Frontend Repo](https://github.com/bhaveshh04/setuai-frontend)
- 🔗 [Backend Repo](https://github.com/bhaveshh04/setuai-backend)

## ✨ Features
- RAG-based contextual question answering
- Animated AI tutor character with realistic lip-syncing
- Microservice-oriented Spring Boot backend
- Dynamic LLM response generation
